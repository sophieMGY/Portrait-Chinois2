Procédure à suivre pour installer le site sur un serveur local XAMPP :

Pour pouvoir installer notre site web sur un serveur local tel que XAMPP, il faut tout d'abord télécharger l'application XAMPP. Ensuite, rendez-vous sur l'onglet "Fichiers" et ouvrez les deux fichiers : XAMPP Start et XAMPP Control.

Une fois les deux fichiers ouverts, accédez au dossier htdocs, et c'est à l'intérieur de ce dossier que nous allons pouvoir déposer le dossier contenant notre site web. Normalement, après avoir effectué toutes ces étapes, nous pourrons lancer notre site à partir de XAMPP.
